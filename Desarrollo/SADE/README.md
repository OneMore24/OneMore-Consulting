# Proyecto: SADE (Sistema de Administración de Datos Empresariales)

**Cliente:** Grupo Andino Comercial
**Estado Actual:** ⚪ Pendiente (Firma de Contrato)

* Alertas de stock mínimo.
* Generación automática de órdenes de compra a proveedores.

* **Lenguaje:** Java 
* **Base de Datos:** Oracle DB
* **Arquitectura:** MVC (Modelo-Vista-Controlador)
