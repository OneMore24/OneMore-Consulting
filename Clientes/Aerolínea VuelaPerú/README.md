# Perfil del Cliente: Aerolínea VuelaPerú

**Sector:** Transporte Aéreo y Turismo.
**Ubicación:** Aeropuerto Jorge Chávez, Lima.

### Descripción
VuelaPerú es una aerolínea regional en expansión. Requieren el sistema **VUELA_PERU** para gestionar la programación de tripulaciones y el mantenimiento preventivo de su flota, asegurando el cumplimiento de las normativas de aeronáutica civil.
