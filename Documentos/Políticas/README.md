# ⚖️ Políticas Corporativas

Este directorio contiene los marcos normativos que rigen el comportamiento ético y legal de **OneMore Consulting**.

### 📄 Contenido
* **Gestión de Datos:** Protocolos para el manejo de información sensible de pacientes y clientes.
* **Confidencialidad (NDA):** Acuerdos de no divulgación para los miembros del equipo y terceros.
* **Reglas de Negocio:** Normas sobre la propiedad intelectual del código y activos digitales.
