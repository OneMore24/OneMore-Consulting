# 🛠️ Directrices Técnicas

Guías operativas para asegurar que el desarrollo de software mantenga un estándar de calidad superior y consistencia técnica.

### 📜 Estándares de Ingeniería
* **Código Limpio:** Convenciones de estilo para **C++** y **SQL**.
* **Control de Versiones:** Guía de *Git Flow* (nombramiento de ramas y estructura de commits).
* **Documentación de Código:** Uso de comentarios técnicos y generación de diccionarios de datos.

> *Estado: En actualización constante por el equipo de arquitectura.*
